# Andor Browser

Andor is a lightweight open-source web browser for Windows built with Python and PyQt.  
It features tabbed browsing, Google homepage support, and a Windows installer.

## Features

- Tabbed browsing
- Integrated navigation controls
- Custom launcher icon
- Windows installer with splash screen
- Fast startup
- Open-source (MIT License)

## Installation

1. Download the latest installer from Releases:
   https://github.com/YOURNAME/Andor/releases

2. Run `AndorSetup.exe`

3. (Optional) Create desktop shortcut during setup
